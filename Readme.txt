Το πρόγραμμα μεταγλωτίζεται μέσω makefile.
Η κλήση του προγράμματος γίνεται με την εντολή
./invoke-oracle [-k NUM] SIZE DEPTH

Μόλις ξεκινήσει το πρόγραμμα ζητάει από το χρήστη ένα seed για την αρχικοποιηση  της oracle.
Αρχικοποιώ κάποιες μεταβλητές και καλώ τη συνάρτηση search με αρχικό όρισμα την κενή συμβολοσειρά.
Η συνάρτηση search είναι η συνάρτηση στη οποία ελέγχω τις λέξεις μου και φτιάχνω το δέντρο.
Δουλέυει αναδρομικά και φτιάχνει το δέντρο με το σκεπτικό της κατα βάθος αναζήτησης.Ελέγχει τις λέξεις που επιστρέφει η oracle και με βάση το bloom filter τις περνάει στο δέντρο ή σε μια δευτερεύουσα δομή λίστας.
Η λιστα αυτή θα ελεγχθεί μετά τη search σε περίπτωση που δεν βρεί τη λύση στο δοσμένο βάθος του δέντρου.
---Έχω αφήσει το πρόγραμμα να εκτυπώνει τις λεξεις που ελέγχει μέσα στη search έτσι ώστε να είναι ξεκάθαρο ότι τρέχει.---
Άν βρεθεί η λέξη μέσα στη search θα εκτυπώσει τη λέξη κάποια στοιχεία που θεώρησα χρήσιμα όπως πόσες λέξεις έχει ελεγξει το βάθος και πόσες λέξεις έχει περάσει στη λίστα.
Μετά θα περιμένει λίγο (sleep(5) ) θα επιστρέψει στη main και θα καλεστεί η συνάρτηση ---print_tree--- η οποία θα εκτυπώσει το δέντρο που έχει φτιαχτεί.
Εάν δεν βρει λέξη θα γυρίσει στη main θα εκτυπωθεί το δέντρο και θα καλεστεί η συνάρτηση search_again η οποία θα πάρει τη λίστα extra 
και θα αρχίσει θα ψάχνει όλες τις λέξεις που δεν πέρασαν το bloom filter και θα προσθέτει όλα τα παιδιά τους στο τέλος αυτής της λίστας.
Δηλαδή κάνει μια αναζήτηση με βάση τη λογική της αναζήτησης κατα πλάτος.
Το bloom filter το αρχικοποιώ:
bf=malloc(size*sizeof(unsigned char));
και κάνω πράξεις με bitwise operators μέσα στη συνάρτηση check(ολισθήσεις και πράξεις and or).
